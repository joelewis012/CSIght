# Code of Conduct

## Our standard

CSIght is a technical project. We keep things simple:

- Be respectful and constructive
- Focus on the problem, not the person
- Help others where you can
- Accept that maintainers have final say on direction

## Not acceptable

- Harassment, abuse, or personal attacks
- Spam or automated noise without prior discussion
- Sharing or using CSIght to violate others' privacy

## Enforcement

Issues or PRs that violate this will be closed. Repeated violations result in being blocked from the repo.

## Scope

This applies to all project spaces — GitHub issues, PRs, discussions, and any other communication related to CSIght.
